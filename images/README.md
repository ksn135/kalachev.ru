# Изображения

Поместите ваше фото в эту папку с именем `photo.jpg`.

Рекомендуемые параметры:
- Формат: JPG или PNG
- Размер: минимум 400x400 пикселей (квадратное изображение)
- Файл должен называться: `photo.jpg` или `photo.png`

Если фото не будет найдено, будет показан placeholder с градиентом.
